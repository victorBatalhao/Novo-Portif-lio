## 🎯 Sobre o Projeto

Este projeto nasceu da necessidade de criar uma **presença digital profissional e marcante** para profissionais da área de TI que desejam se destacar no mercado. O portfólio foi concebido com três pilares fundamentais:

> 🧠 **Clareza** — Informação bem organizada que comunica valor rapidamente  
> 🎨 **Impacto Visual** — Design moderno que impressiona recrutadores e clientes  
> ⚡ **Performance** — Zero dependências externas, carregamento instantâneo

### 🎪 Intuito do Portfólio

O portfólio serve como seu **cartão de visita digital**, mas muito além disso — é uma vitrine interativa que demonstra não apenas o que você sabe fazer, mas **como você pensa e apresenta** seu trabalho. Ele foi desenhado para:

- ✅ Impressionar recrutadores técnicos e gestores de contratação
- ✅ Centralizar sua presença profissional (GitHub, LinkedIn, Credly, Email)
- ✅ Demonstrar habilidades front-end de forma implícita pelo próprio design
- ✅ Funcionar como landing page para propostas e oportunidades freelance
- ✅ Ser facilmente encontrado e compartilhado através de um link único na Vercel

## 🛠️ Tecnologias Utilizadas

```
📦 Stack tecnológica
├── 🌐 HTML5          — Semântica moderna, acessibilidade e SEO
├── 🎨 CSS3           — Custom Properties, Grid, Flexbox, Animations
├── ⚡ JavaScript ES6+ — IntersectionObserver, FileReader API, DOM
├── 🔤 Google Fonts   — Syne (display) + DM Sans (corpo) + Space Mono
└── 🚀 Vercel         — Hospedagem e deploy contínuo
```

> **Por que sem frameworks?** A escolha de HTML/CSS/JS puro garante **velocidade máxima de carregamento**, **zero dependências** para atualizar e **compatibilidade universal** com qualquer browser moderno — essencial para a primeira impressão de um recrutador.

## ✏️ Personalização

### 🧑 Suas informações pessoais

Abra `index.html` e localize as marcações com comentários para editar:

```html
<!-- SEU NOME -->
<h1 class="profile-name" contenteditable="true">Seu Nome Aqui</h1>

<!-- SEU CARGO -->
<p class="profile-title" contenteditable="true">Seu Cargo · Sua Especialidade</p>

<!-- SUAS ESTATÍSTICAS -->
<span class="stat-num" id="statProjects">12</span>   <!-- nº de projetos -->
<span class="stat-num" id="statCerts">8</span>        <!-- nº de certificações -->
<span class="stat-num" id="statYears">4</span>        <!-- anos de experiência -->
```

### 🔗 Seus links

Substitua todas as ocorrências de `seu-usuario` e `seu@email.com`:

```html
<!-- GitHub -->
<a href="https://github.com/seu-usuario" ...>

<!-- LinkedIn -->
<a href="https://www.linkedin.com/in/seu-usuario" ...>

<!-- Email -->
<a href="mailto:seu@email.com" ...>

<!-- Credly -->
<a href="https://www.credly.com/users/seu-usuario" ...>
<a href="https://www.credly.com/badges/SEU-BADGE-ID" ...>
```

### 🎨 Cores personalizadas

As cores estão centralizadas em variáveis CSS no início de `style.css`:

```css
:root {
  --blue-600: #1d4ed8;   /* cor principal dos botões */
  --blue-400: #3b82f6;   /* destaques e hover */
  --cyan-400: #22d3ee;   /* acentos e bordas */
  /* Troque por qualquer paleta que preferir */
}
```

### 🛠️ Suas habilidades

Edite os cards de habilidades em `index.html` e atualize os percentuais:

```html
<div class="skill-bar-item">
  <span>React</span>
  <div class="skill-bar">
    <div class="skill-fill" data-width="88%"></div>  <!-- altere o % -->
  </div>
  <span>88%</span>
</div>
```

### 📦 Seus projetos

Cada card de projeto segue este padrão — copie e edite quantos precisar:

```html
<article class="project-card">
  <div class="project-card-inner">
    <!-- Cor da linguagem (ex: #f7df1e para JS, #3572A5 para Python) -->
    <div class="project-lang-dot" style="--clr:#f7df1e"></div>

    <div class="project-header">
      <div class="project-icon">📦</div>
      <div class="project-links">
        <a href="https://github.com/seu-usuario/nome-do-repo" target="_blank" class="icon-link">
          <!-- ícone GitHub -->
        </a>
        <a href="https://seu-projeto.vercel.app" target="_blank" class="icon-link">
          <!-- ícone de link externo -->
        </a>
      </div>
    </div>

    <h3 class="project-title">Nome do Projeto</h3>
    <p class="project-desc">Descrição clara e objetiva do que o projeto resolve.</p>
    <div class="project-tech">
      <span>React</span><span>Node.js</span><span>PostgreSQL</span>
    </div>
  </div>
</article>
```

> 💡 **Dica:** Adicione a classe `featured` ao card mais importante para ele ocupar largura dupla na grid.

## 🌐 Deploy na Vercel

### Método 1 — Via GitHub (recomendado)

```bash
# 1. Crie um repositório no GitHub e envie os arquivos
git init
git add .
git commit -m "feat: portfólio inicial"
git branch -M main
git remote add origin https://github.com/seu-usuario/portfolio.git
git push -u origin main

# 2. Acesse vercel.com → "Add New Project"
# 3. Conecte sua conta GitHub
# 4. Selecione o repositório → Deploy (sem nenhuma configuração!)
```

### Método 2 — Via Vercel CLI

```bash
npm i -g vercel
vercel
# Siga as instruções interativas
```

### ⚙️ Configuração recomendada na Vercel

```json
// vercel.json (opcional — para domínio customizado e redirects)
{
  "rewrites": [{ "source": "/(.*)", "destination": "/index.html" }],
  "headers": [
    {
      "source": "/(.*)",
      "headers": [
        { "key": "X-Content-Type-Options", "value": "nosniff" },
        { "key": "X-Frame-Options", "value": "DENY" }
      ]
    }
  ]
}
```

> Após o deploy, a Vercel fornecerá um domínio `seu-projeto.vercel.app` e você pode configurar um domínio customizado gratuitamente.

## 💡 Boas Práticas

### 📝 Conteúdo que converte

- **Seja específico nos projetos** — "Aumentei performance em 40%" > "Otimizei o sistema"
- **Use números reais** nas estatísticas do perfil (projetos, certificações, anos)
- **Escreva a descrição de cargo para a vaga que você QUER**, não apenas a que você tem
- **Mantenha no máximo 4 projetos** — curadoria é mais impactante que quantidade

### 🔍 SEO básico

```html
<!-- Adicione ao <head> do index.html -->
<title>Seu Nome — Desenvolvedor Full Stack | Portfólio</title>
<meta name="description" content="Desenvolvedor Full Stack com X anos de experiência em React, Node.js e AWS." />
<meta name="keywords" content="desenvolvedor, full stack, react, node.js, portfólio" />
<link rel="canonical" href="https://seu-site.vercel.app" />
```

### ⚡ Performance

- Otimize sua foto de perfil antes de fazer upload (recomendado: **WebP, máx. 200KB**)
- Use o [PageSpeed Insights](https://pagespeed.web.dev/) para verificar sua pontuação
- O portfólio foi construído para atingir **90+ no Lighthouse** por padrão

### 🔐 Segurança

- Nunca exponha tokens de API diretamente no código front-end
- Se futuramente adicionar um formulário de contato, use um serviço como Formspree ou um backend
- Mantenha o `vercel.json` com os headers de segurança recomendados
